# Kafee

Let's get energized with Kafka!

## Goals

- [ ] Encapsulate message publishing
- [ ] Open telemetry support
- [ ] Retries
- [ ] Adaptable message encoding
  - [ ] JSON
  - [ ] Avro
  - [ ] Protobuf

## Installation

Just add [`kafee`](https://hex.pm/stordco/kafee) to your `mix.exs` file like so:
